# WebApplicationBuilder

`builder`. Creamos un `builder`:

```csharp
static void Main(string[] args)
{
    WebApplicationBuilder builder = WebApplication.CreateBuilder(args);
    WebApplication app = builder.Build();
    app.Run();
}
```

Esto creao una aplicación web. No tiene ninguna función ni nada que responder.
Pero es una instancia de un servidor web.

Middleware. Se especifica _entre_ la creación de `app` y la sentencia
`app.Run()`.

```csharp
    // ...
    WebApplication app = builder.Build();

    // Use(context, next) es un método Middleware para especificar el
    // comportamiento de nuestro servidor web.
    app.Use(async (context, next) =>
    {
        Console.WriteLine("Solicitud recibida");
        // Llama al siguiente Middleware, que sería el Run(context) de abajo
        await next();
    });

    // Run(context) es otro método Middleware que permite devolver una respuesta
    // desde el servidor.
    app.Run(async context =>
    {
        await context.Response.WriteAsync("hola mundo");
    });

    app.Run();
    // ...
```

Crear aplicación web con `dotnet`.

```sh
dotnet new webapi --use-controllers --use-program-main --no-openapi -o Nombre
```

Esto crea el siguiente _template_.

```csharp
public static void Main(string[] args)
{

}
```

# Modelo

Creamos carpeta `Models`. En la carpeta raíz del proyecto.

Creamos allí `ApplicationDbContext.cs`. Creamos una clase, mismo nombre del
archivo.

```csharp
using Microsoft.EntityFrameworkCore;

public class ApplicationDbContext : DbContext
{

}
```

Creamos clase _POCO_ `Pelicula`.

```csharp
public class Pelicula
{
    public int PeliculaId { get; set; }
    public string Titulo { get; set; }
    public string Genero { get; set; }
    public DateTime FechaEstreno { get; set; }
    public Director Director { get; set; }
}
```

Creamos la clase _POCO_ `Director`:

```csharp
public class Director
{
    public int DirectorId { get; set; }
    public string Nombre { get; set; }
    public int Edad { get; set; }
    public List<Pelicula> Peliculas { get; set; }
}
```

Volvemos al `DbContext`.

```csharp
using Microsoft.EntityFrameworkCore;

public class ApplicationDbContext : DbContext
{
    public DbSet<Pelicula> Peliculas { get; set; }
    public DbSet<Director> Directores { get; set; }

    public ApplicationDbContext()
    {

    }
}
```

En el archivo principal, en el Middleware, especificamos el `DbContext`.

```csharp
builder.Services.AddControllers();
builder.Services.AddDbContext<ApplicationDbContext>(options =>
    // Para agregar SQLServer, también podemos usar SQLite (revisar clase)
    options.UseSqlServer("Server=localhost\\SQLEXPRESS; Database=Peliculas; Trusted_Connection=True")
);
```
