# Controladores

Controladores. Permiten conectar bases de datos relacionales con lenguajes de
programación orientados a objetos. Se puede abstraer a distintos niveles.
Podemos escribir consultas `SQL` directamente en `C#` o trabajar directamente
en _orientación a objetos_ y que el lenguaje se encargue de la prelación con la
base de datos.

# NUGET

Repositorio de paquetes de `.NET`. Usamos:

`dotnet package add <paquete>`

Instala paquetes en `$HOME/.nuget`. Ahí podemos revisar el código del paquete.

`nuget.org`. Front-end web del repositorio. Podemos obtener información sobre
los paquetes ahí.

# Dapper

Es una biblioteca ORM (Object-Relational Mapping, mapeador objeto-relación)
simple. Permite vincular bases de datos relacionales con un lenguaje de
programación orientado a objetos como es `C#`.

Incorporamos Dapper:

`dotnet package add Dapper`

Esto no cambia el código. Pero cambia la definición de nuestro proyecto. Ahora
hace referencia a Dapper.
