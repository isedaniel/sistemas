# Entity Framework

Creamos el proyecto. En el proyecto hacemos una clase `Pelicula`.

```csharp
public class Pelicula
{
    public int ID { get; set; }

    public string Titulo { get; set; }

    public string Genero { get; set; }
}
```

Y en el método `Main` tomamos _input_ del usuario.

```csharp
static void Main(string[] args)
{
    int id = Convert.ToInt32(Console.ReadLine());
    string titulo = Console.ReadLine();
    string genero = Console.ReadLine();

    Pelicula pelicula = new Pelicula();
    pelicula.ID = id;
    pelicula.Titulo = titulo;
    pelicula.Genero = genero;
}
```

Agregamos `Entity Framework`:

```powershell
dotnet package add Microsoft.EntityFrameworkCore.Sqlite
```

Esto incorpora `<ItemGroup><PackageReference Include=...>` al archivo `.csproj`
del proyecto y por lo tanto el paquete `Entity Framework` puede ser llamado
desde nuestro proyecto.

Instalamos una herramienta para poder llamar a `dotnet-ef`:

```powershell
dotnet tool install --global dotnet-ef
```

# Clase DBContext

Para usar Entity Framework creamos una clase que herede de `DbContext`:

```csharp
public class ApplicationDbContext : DbContext
{
    public DbSet<Pelicula> Peliculas { get; set; }

    protected override void OnConfiguring(DbContext options)
    {
        options.UseSQLite;
    }

}
```
