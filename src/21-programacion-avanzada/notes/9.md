# jsonplaceholder

Solicitudes. Vamos a enviarlas a este sitio. Responde en formato `JSON`. El
objetivo es aprender a enviar solicitudes y recibir JSON en nuestro programa en
C#.

`jsonplaceholder.typicode.com`

# Enviar solicitudes

Creamos entidad `HttpClient`:

```csharp
HttpClient httpClient = new HttpClient();
```

De esta forma la aplicación solicita al sistema operativo recursos para enviar
la solicitud. Cada vez que usamos recursos del sistema tenemos que pensar en
liberarlos. En este caso usamos _SOCKETS_.

Para eso usamos un bloque `using`. Equivale a un bloque `try-finally`, donde en
el bloque `finally` llamaríamos a `httpClient.Dispose();`. Eso es lo que hace
`using`. Para poder usar una clase en una estructura `using` tiene que
implementar la interfaz `IDisposable`, que pide implementar el método
`void Dispose()`

Hacemos la solicitud:

```csharp
// Seteamos la url
string url = "https://jsonplaceholder.typicode.com";

// Creamos objeto Uri para la solicitud
Uri uri = new Uri(url);

// Seteamos BaseAddress a la Uri
httpClient.BaseAddress = uri;

// Hacemos la solicitud con un método asicrónico
// Lo esperamos con await
// Solo se puede usar en método asincrónicos
HttpResponseMessage respuesta = await httpClient.GetAsync("posts");
string json = await respuesta.Content.ReadAsStringAsync();

// Imprimimos el texto del JSON
console.WriteLine(json);
```

Podemos chequear si la solicitud tuvo éxito:

```csharp
HttpResponseMessage respuesta = await httpClient.GetAsync("posts");

try
{
    // Si el código no fue exitoso, tira una excepción que captura el catch
    respuesta.EnsureSuccessStatusCode();
    string json = await respuesta.Content.ReadAsStringAsync();
    console.WriteLine(json);
}
catch (Exception ex)
{
    Console.WriteLine($"Error al procesar la solicitud: {ex.Message}");
}
```

# Traducir JSON a objetos POCO

Propiedades JSON. En este caso, cada JSON tiene cuatro propiedades:
- userID
- 

Clase POCO. Modelamos una clase POCO en el archivo `Post.cs`:

```csharp
public class Post
{
    // las propiedades tienen que coincidir case-sensitive con las claves JSON
    public int userID { get; set; }
    public int id { get; set; }
    public string title { get; set; }
    public string body { get; set; }
}
```

Deserializar. Convierte una respuesta `JSON` a una instancia de una `POCO`.

```csharp
HttpResponseMessage respuesta = await httpClient.GetAsync("posts");

try
{
    // Si el código no fue exitoso, tira una excepción que captura el catch
    respuesta.EnsureSuccessStatusCode();
    string json = await respuesta.Content.ReadAsStringAsync();

    // Creamos una lista de Posts basándonos en el string JSON
    List<Post> posts = JsonSerializer.Deserialize<List<Post>>(json);

    // Imprimimos
    foreach (var post in posts)
    {
        Console.WriteLine($"Título: {post.title}");
    }
}
catch (Exception ex)
{
    Console.WriteLine($"Error al procesar la solicitud: {ex.Message}");
}
```

# Equivalente del CREATE

```csharp
// Creamos un POST
Post post = new Post();
post.userId = 1;
post.title = "Título";
post.body = "Cuerpo;

// Traducimos el POCO a JSON
// JsonContent además lo incluye en un paquete HTTP
var content = JsonContent.Create(post);

// Hacemos la solicitud, agregando el content
HttpResponseMessage respuesta = await httpClient.PostAsync("/posts", content);

try
{
    // Si el código no fue exitoso, tira una excepción que captura el catch
    respuesta.EnsureSuccessStatusCode();
    // Imprimimos mensaje de éxito
    Console.WriteLine("Recurso creado con éxito.");
}
catch (Exception ex)
{
    Console.WriteLine($"Error al procesar la solicitud: {ex.Message}");
}
```

# Actualización de un recurso

Usamos método `PUT`. Para modificarlo primero lo obtenemos:

```csharp
HttpResponseMessage respuesta = await httpClient.GetAsync("posts/1");

try
{
    // Primero pedimos un post
    respuesta.EnsureSuccessStatusCode();
    string json = await respuesta.Content.ReadAsStringAsync();

    // Deserializamos
    Post post = JsonSerializer.Deserialize<Post>(json);

    post.title = "New title";
    var content = JsonContent.Create(post);
    var respuestaPut = await httpClient.PutAsync("posts/1", content);

    // Chequeamos que haya llegado bien
    respuestaPut.EnsureSuccessStatusCode();

}
catch (Exception ex)
{
    Console.WriteLine($"Error al procesar la solicitud: {ex.Message}");
}
```
