# Comando para crear proyecto

Crear con clase `Program` y método `Main`:

```
dotnet new console -O Name --use-program-main
```

# Espacio de nombre

Las clases van dentro del `namespace` del proyecto.

```c
namespace NombreProyecto;
```

Va uno solo por archivo. Una clase por archivo. Vamos a usar uno solo para
todo el proyecto.
