# Trigger

Un _disparador_, es un tipo de _stored procedure_. Un código que la base de
datos pueda interpretar.
Se ejecuta frente a una condición, por ejemplo,
cuando intento modificar (agregar, actualizar, eliminar) datos en una tabla.

Su función es conservar integridad y coherencia de datos.

Por ejemplo: toda información con más de 30 días se traslada a otra base de
datos.
