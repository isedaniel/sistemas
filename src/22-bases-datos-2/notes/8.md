# MongoDB

Sistema de bases de datos multiplataforma orientado a documentos, de esquema
libre. Desarrollada en C++. Alta velocidad.

Colecciones. Equivalente a las tablas en una base de datos relacional.

Documentos. Equivalente a los registros de una base de datos relacional. Se
agrupan en colecciones. Se almacena en `BSON`, formato _Binary JSON_.

BSON. _Binary_ JSON. Ocupa más espacio que JSON. Es importante que
almacenamiento sea barato para que NoSQL sea conveniente.

JSON. JavaScript Object Notation. Formato ligero. Basado en un subconjunto del
lenguaje `JavaScript`. Es simple. Se emplea donde el volumen de flujo de datos
es importante. No requiere esquemas ni tablas. Más simple para los equipos de
desarrollo porque es un formato familiar.
