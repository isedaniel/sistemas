# PowerBI

Sistema de _Business Intelligence_. Toma datos de múltiples fuentes.

# Poder de PowerBI

- Desucbrir información.
- Conectar con muchos orígenes de datos.
- Calcular a nivel de campo con lenguaje DAX, sintaxis similar a Excel.
- Creación de relaciones en datasets.

# Arquitectura

PowerBI Desktop. Publica en Power BI Service. Esto se sostiene con Power BI
Gateways. Desde ahí otros usuarios pueden acceder a la información.

Uno puede conectar datos de otros orígenes. SQL Server. Archivos de texto plano.
Esa información es consumida por unas _Vistas_. Eso se sincroniza con la nube.
Una vez en la nube, puede ser visualizada por los usuarios autorizados.

# Power BI Desktop

Para desarrollar y publicar los reportes. Aquí se hace el modelado de datos.

# Power BI Service

Instancia en la nube donde se publican los reportes al App Workspace. Administra
permisos y dashboards.
