# Condiciones

Lugar cómodo, sin ruido. Los ejercicios se hacen a mano, escribiendo. Se saca
foto y se sube. Cada paso se justifica teóricamente. Los ejercicios van a ser
distintos por persona. Hay un conjunto de ejercicios. El sistema asigna de
forma aleatoria. Cada examen va a ser distinto.
