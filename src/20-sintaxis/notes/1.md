# Lenguajes naturales

Cada elemento en un lenguaje natural tiene un significado. Incluidas imágenes.

Características.

1. Son ambiguos. Tanto en significado (semántica) como en gramática.

2. El significado es más importante que la sintaxis: hasta un mensaje mal
   escrito (en términos sintácticos) puede llegar a entenderse bien.

3. Evolucionan con el tiempo.

4. Las reglas aparecen después, para sistemtizar el uso del lenguaje.

En esta materia no vamos a estudiar los lenguajes naturales.

# Lenguajes formales

Estos lenguajes vamos a estudiar.

Características.

1. No pueden ser ambiguos.

2. Solo tienen sintaxis. No poseen semántica: sintaxis > semántica

3. No evolucionan. Un cambio en un lenguaje formal implica un cambio de
   versión: estamos ante un lenguaje distinto.

4. Primero se definen las reglas $\to$ luego se define el lenguaje.

La matemática es un ejemplo. Un lenguaje de programación es otro.

# Símbolo

**Símbolos.** Elementos básicos. Son fundamentales e indivisibles. Forman
alfabetos, por ejemplo: $/$, $A$, $\Lambda$.

# Alfabeto

**Alfabeto.** Conjunto finito, no vacío, no ordenado, de símbolos, por ejemplo:

$\Sigma_1 = \{a, b, \dots z\}$

$\Sigma_2 = \{1, 2, \dots 9, 0\}$

$\Sigma_3 = \{ /, *, +, - \}$

(Un conjunto es una lista no ordenada, por definición, salvo que se aclare lo
contrario)

Cada símbolo es un elemento. Si tenemos el alfabeto $\Sigma = \{ a, b, c \}$,
$a$ pertenece. Pero $A$ no.

Además, combinaciones de símbolos no pertenecen al alfabeto: $ab$ no pertenece.

# Operaciones con símbolos

Potenciación. Repetimos el símbolo tantas veces como indique el exponente. Por
ejemplo:

- $a^0 = \lambda$

- $a^1 = a$

- $a^3 = aaa$

- $z^2t^6c^2o^4 = zzttttttccoooo$

# Palabra

Cuando concatenamos símbolos del alfabeto obtenemos *palabras*.

Palabra. Secuencia finita de símbolos. Dado un alfabeto, puedo formar infinitas
palabras.

*Palabra vacía:* se simboliza $\lambda$.

# Operaciones con palabras

*Longitud.* Cantidad de símbolos que tiene:

- $|perro| = 5$

- $|\lambda| = 0$

*Concatenación.* Unimos dos palabras para formar otra.

- $Buenos \cdot Aires = BuenosAires$

- $Perro \cdot \lambda = Perro$

La concatenación es asociativa. No es conmutativa.

*Potenciación.* Concatenar una palabra consigo misma, la cantidad de veces que
indique el exponente.

- $Bla^3 = BlaBlaBla$

- $Perro^0 = \lambda$

*Inversión, reflexión, transposición.* Inversión del orden de los símbolos que
componen una palabra.

- $arbol^{-1} = lobra$

Si invierto de nuevo, vuelvo a la palabra original:

- $lobra^{-1} = arbol$

Eso significa que la *inversión* es *involutiva*.

# Lenguaje universal, o clausura de Kleene

*Lenguaje universal, clausura de Kleene.* Conjunto de todas las combinaciones
posibles de símbolos, dado un alfabeto. Es decir, conjunto de todas las
palabras que se pueden formar, dado un alfabeto.

Las combinaciones posibles de símbolos son infinitas. Por ello, la clausura de
Kleene es infinita.

Se simboliza $\Sigma^*$ o $L^*$.

# Lenguaje

*Lenguaje.* Es un subconjunto del lenguaje universal, por ejemplo, palabras
reservadas y reglas para nombrar variables.

# Operaciones con lenguajes

*Unión.* Produce un nuevo lenguaje, formado por las palabras del primero y las
del segundo.

- $L_1 \cup L_2 = \{aa, ab\} \cup \{bb, ba\} = \{aa, ab, bb, ba\}$

Es conmutativa. Es asociativa. Tiene elemento neutro ($\Phi$).

*Concatenación.* Concateno cada palabra de un lenguaje con cada palabra de
otro, generando un lenguaje nuevo.

- $L_1 \cdot L_2 = \{aabb, abbb, abbb, abba\}$

Es asociativa. No es conmutativa.

(Note: siempre tener en cuenta con qué estamos trabajando: símbolos, alfabetos,
palabras, lenguajes).

*Potenciación.* Concatena un lenguaje consigo mismo _n_ veces.

- $L_1^2 = \{aa, ab\} \cdot \{aa, ab\} = \{aaaa, aaab, abaa, abab\}$
